# Shipping Guide Screenshots

ضع لقطات الشاشة المطلوبة في هذا الفولدر، بنفس الأسماء المذكورة في `shipping-guide.html` (Section 9).

## القائمة المطلوبة

| الاسم | المحتوى |
|---|---|
| `shipping-card-ready.png` | كارت أوردر في حالة "⏸ ready" مع زر "📦 سجّل الشحن" |
| `ship-setup-modal.png` | الـ modal الكامل لتسجيل الشحن (طريقة + شركة + تكلفة + رقم تتبع) |
| `followup-cards.png` | كروت متابعة (pickup + company + متأخر + جاهز للأرشفة) |
| `settle-modal.png` | modal التسوية المتعددة من حسابات الشحن |
| `accounts-company-card.png` | كارت شركة في حسابات الشحن (KPIs + أوردرات جاهزة) |
| `return-modal.png` | modal تسجيل المرتجع |
| `self-heal-banner.png` | banner الإصلاح الذاتي الأحمر فوق صفحة حسابات الشحن |

## كيف تأخذ لقطة شاشة

- **Windows**: اضغط `Win + Shift + S`
- **Mac**: اضغط `Cmd + Shift + 4`
- **Browser DevTools**: F12 → device toolbar → Screenshot

## نصائح

- استخدم mode dark (الافتراضي) للحفاظ على الـ contrast
- خذ اللقطة في عرض ~1024px للوضوح
- لو في معلومات حساسة (أرقام تليفون / أسعار حقيقية) → blur قبل الحفظ
- فورمات PNG مفضّل للوضوح (JPG لو الحجم مهم)
