🔄 الـ flow الكامل (Automated)
1. الموظف بيبعت اقتراح من زر 💡
2. AI Gemini بيحلل ويولّد pros/cons/خطة
3. أنت بتراجع وتدوس ✅ "قبول"
4. دوس 🤖 "ابعت لـ Claude" → Issue يتنشأ تلقائياً
5. GitHub Action بيشغّل Claude — يكتب الكود ويفتح PR
6. دوس "🔄 تحقق من PR" — لما يبان، افتحه وراجع
7. مرّج الـ PR على main من GitHub
⚙️ Setup Guide — تفعيل التنفيذ التلقائي
💡 محتاج تضبط 2 secrets مرة واحدة. بعد كده كل اقتراح بيتنفذ تلقائياً.
١
احصل على Anthropic API key
١. روح console.anthropic.com → سجّل دخول.
٢. Billing → ادفع $5-10 (للاختبار).
٣. API Keys → Create Key → سمّيه "b2c-erp" → انسخ المفتاح.
٢
ضيفه في GitHub Secrets
روح صفحة Secrets في GitHub:
- Name: ANTHROPIC_API_KEY
- Secret: [المفتاح من الخطوة ١]
- Add secret.
٣
فعّل GitHub Actions PR creation
Settings → Actions → General
في "Workflow permissions":
✅ Read and write permissions
✅ Allow GitHub Actions to create and approve pull requests
٤
احصل على GitHub Personal Access Token
١. روح صفحة Tokens (اللينك بيختار الـ scopes تلقائي).
٢. Expiration: 1 year.
٣. تأكد أن repo scope مفعّل.
٤. Generate token → انسخه.
٥
ضيفه في Firebase Functions Secrets
من الـ terminal على جهازك:
firebase functions:secrets:set GITHUB_PAT هيطلب يلصق الـ token → الصقه → Enter.
بعدها:
firebase deploy --only functions:createSuggestionIssue,functions:checkSuggestionPR
✅ خلاص! بعد كده ضغطة زر "🤖 ابعت لـ Claude" بتعمل كل حاجة تلقائي.
⏳
جاري التحميل...
🤖 الخطوات التالية لإرسال الاقتراح لـ Claude
✅ اتفتح tab جديد لـ GitHub Issue فيه كل تفاصيل الاقتراح + تحليل AI جاهز.
📝 اعمل الخطوات دي بالترتيب:
١
في الـ tab اللي اتفتح (GitHub)
دوس على "Submit new issue" الأخضر في الأسفل. الـ Issue هيتنشأ بـ label claude-task.
٢
روح لـ Claude Code
افتح claude.ai/code → دوس "New session".
٣
اربط بالـ Issue
اختار الـ repo engmah72-star/b2c-erp ولصق رابط الـ Issue اللي عملته في الخطوة ١. Claude هيقرا الـ Issue وكل تحليل AI الموجود فيها.
٤
انتظر Claude يخلص
Claude هيكتب الكود ويعمل PR على فرع جديد + يعلق على الـ Issue برابط الـ PR. مدة التنفيذ من 2-10 دقايق حسب التعقيد.
٥
راجع ومرّج
افتح الـ PR، راجع التغييرات، اعمل merge على main. الموقع يتحدث في خلال دقايق.
💡 ملاحظة: لو الـ Issue معقدة أو محتاجة توضيح، Claude هيعلق بأسئلة. ارجع للـ Issue وارد عليه ليكمل التنفيذ.
