🔥 دليل رفع Business2Card على Firebase Hosting

خطوات بسيطة — مرة واحدة وخلاص
1
تثبيت Node.js
مطلوب أول مرة فقط
روح nodejs.org وحمّل الإصدار LTS — ثم ثبّته.
💡 بعد التثبيت افتح Command Prompt وتأكد: node --version تظهر رقم
2
تثبيت Firebase CLI
مطلوب أول مرة فقط
افتح Command Prompt أو PowerShell واكتب:
npm install -g firebase-tools
⏳ هياخد دقيقتين — انتظر لحد ما يخلص
3
تسجيل الدخول بحساب Google
مطلوب أول مرة فقط 
firebase login
هيفتح المتصفح — سجّل دخول بنفس الحساب اللي عليه Firebase.
4
جهّز مجلد المشروع
مطلوب
عمل مجلد اسمه b2c-erp على سطح المكتب أو أي مكان.
حط فيه الملفات دي كلها:
📄 firebase.json
📄 .firebaserc
🌐 index.html
🔒 login.html
👤 clients.html
✏️ design.html
🏭 production.html
🖨️ print.html
🚚 shipping.html
💰 accounts.html
▣ suppliers.html
📊 reports.html
⚙️ settings.html
🎨 shared.css
⚠️ الملفات كلها في نفس المجلد — مش في مجلدات فرعية
5
ارفع على Firebase
المطلوب
افتح Command Prompt جوه المجلد:
# اول حاجة — افتح CMD في المجلد
# كليك يمين في المجلد ← "Open in Terminal"

# ثم اكتب:
firebase deploy --project business2card-c041b
✅ بعد دقيقة هيطلعلك رابط زي ده:
Hosting URL: https://business2card-c041b.web.app
6
إضافة بيانات المستخدم في Firestore
مهم جداً
بعد الرفع — روح Firebase Console وعمل الآتي:
1️⃣ روح Authentication → Users — هتلاقي الإيميل بتاعك والـ UID
2️⃣ روح Firestore Database → users collection
3️⃣ اضغط "Add document"
4️⃣ في Document ID اكتب نفس الـ UID بالظبط
5️⃣ أضف الـ fields دي: 
name    → "محمود حسين"
email   → "your@email.com"
role    → "admin"
isActive → true
✅ بعد كده تفتح الموقع وتدخل بالإيميل وكلمة المرور — هيشتغل تمام
7
Firestore Rules (لو لسه مش مضبوطة)
اختياري
روح Firestore → Rules وحط الكود ده:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
💡 ده بيسمح لأي مستخدم مسجل دخول يقرأ ويكتب
🎉 السيستم شغال!
الرابط النهائي:
https://business2card-c041b.web.app
لما تحدث أي ملف — ارجع اكتب firebase deploy وخلاص
