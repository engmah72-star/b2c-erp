# 🔐 إعداد Firebase Security Rules

## الخطوات

1. افتح: https://console.firebase.google.com
2. اختر مشروع: business2card-c041b
3. من القائمة اليسرى: Firestore Database
4. اضغط على tab: Rules
5. احذف كل الـ rules الموجودة
6. انسخ محتوى ملف `firestore.rules` والصقه
7. اضغط: Publish

## ملاحظة
هذه الـ Rules تحمي البيانات على مستوى Firebase
وليس فقط على مستوى الكود.
