## 🚀 خطواتك الأولى (لأي دور)

### 1️⃣ تسجيل الدخول
1. افتح الرابط على متصفحك (أو خد من admin لو مش متذكّر)
2. سجّل **رقم موبايلك** (بدون كود الدولة) في خانة الإيميل
3. سجّل **كلمة السر** اللي اعطاها لك admin
4. أول مرة → النظام يطلب منك تغيير كلمة السر — اعمل ده فوراً

### 2️⃣ صفحتك الرئيسية
كل دور بينزّل على **لوحته الخاصة**:
- خدمة العملاء → `cs-dashboard.html`
- المصمم → `designer-dashboard.html`
- التنفيذ → `production-dashboard.html`
- الشحن → `shipping-dashboard.html`
- المحاسب / Admin → `accounts.html`
- مدير التشغيل → `ops-dashboard.html`

### 3️⃣ الـ Sidebar
على الشمال (في وضع desktop) أو زر "≡" (في الموبايل):
- 🏠 **لوحتي** = الصفحة الرئيسية لدورك
- 👥 **عملاء** = قائمة العملاء وأوردراتهم
- 💬 **inbox** = الرسائل والمحادثات
- 🔔 **إشعارات** = آخر التحديثات الموجَّهة لك
- ⚙️ **بروفايلي** = بياناتك + كلمة السر

## 📱 PWA — تنزيل النظام كـ App

### iPhone (Safari)
1. افتح النظام في Safari
2. اضغط زر **مشاركة** (المربع مع السهم لفوق)
3. اختار **"إضافة إلى الشاشة الرئيسية"**
4. اعتمد → الـ app يظهر مع icon على شاشتك مباشرة

### Android (Chrome)
1. افتح النظام في Chrome
2. اضغط **⋮** (النقط الثلاث فوق)
3. اختار **"إضافة إلى الشاشة الرئيسية"** أو **"Install app"**
4. اعتمد → تماماً زي أي app

### ميزات الـ PWA
- بيشتغل **offline** للصفحات اللي زرتها قبل كده
- بيدفع لك **إشعارات** على الموبايل
- بيبدأ أسرع لأن CSS/JS متخزّن محلياً

## 🆘 لو حصلت مشكلة

| المشكلة | الحل |
|---------|------|
| نسيت كلمة السر | اطلب من admin أو مدير التشغيل يعيد التعيين من بروفايلك |
| الـ app بيقع أو بيظهر صفحة بيضاء | اعمل **refresh** كامل (Ctrl+Shift+R أو "Pull to refresh" على الموبايل) |
| لو الـ refresh لم يحل المشكلة | افتح `reset-sw.html` لمسح الـ cache الكامل |
| فيه أوردر بياناته غلط | كلّم admin أو مدير التشغيل — فيه audit log كامل لكل تعديل |
| رقم تليفون عميل مش بيظهر | ده طبيعي حسب دورك — أدوار التصميم/التنفيذ مش بيشوفوا أرقام العملاء (RULE 8) |
| الـ زر مش شغّال | لو ضغطت ومفيش حاجة بتحصل → جرّب refresh، لو استمر → كلّم admin |

## 📜 قواعد ذهبية لكل الأدوار

1. **سجّل خروج** لمّا تخلّص شغل (خصوصاً على جهاز مشترك)
2. **متشاركش كلمة سرّك** — كل واحد له حساب خاص
3. **متعملش double-click** على أي زر — النظام محمي ضد ده لكن تجنّبه
4. **لو شفت رسالة "⚠️"** بـ تفسير → اقراها بعناية قبل ما تكمل
5. **كل حركة مالية لها audit trail** — مدير التشغيل أو الأدمن يقدر يراجع كل تعديل
