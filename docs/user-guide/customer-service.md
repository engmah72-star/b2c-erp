# 💬 دليل خدمة العملاء (Customer Service)

> دورك هو **قلب التواصل** مع العملاء — تستقبل الطلبات، تتابع الأوردرات، تسجّل الدفعات، تكلم العميل لمّا الأوردر يجهز.

---

## 🏠 لوحتك الرئيسية — `cs-dashboard.html`

أول حاجة تظهر لك يومياً:

### الأقسام الأساسية
- **🔍 بحث فوري (Ctrl+K)** — ابحث عن عميل/تليفون/أوردر فوراً
- **اليوم بالأرقام** — كم عميل/أوردر/تحصيل لك النهارده
- **💬 رسائل لم يُرد عليها** — من inbox
- **🚚 جاهز للتسليم** — أوردرات جاهزة عند الشحن، اتصل بالعميل
- **📋 قائمة المتابعة** — متابعات معلّقة (calls, WhatsApp, زيارات)
- **🗂 أوردراتي** — تبويب نشطة / باقي / منجزة

### تسجيل الحضور
- زر **"تسجيل حضور / انصراف"** أعلى يمين الصفحة
- اعمل تسجيل دخول/خروج كل يوم — مدير التشغيل بيتابع

## 👤 العملاء — `clients.html`

### إنشاء عميل جديد
1. زر **"+ عميل جديد"**
2. املأ: الاسم، تليفون1 (إجباري)، تليفون2، الإيميل (اختياري)
3. اختار المحافظة + المدينة
4. **سيكشن "أوردر جديد"** يفتح فوراً بعد الحفظ — اعمل الأوردر بنفس الـ flow

### إنشاء أوردر جديد لعميل موجود
1. ابحث عن العميل (شريط البحث)
2. اضغط على بطاقته → يفتح panel على اليمين
3. زر **"➕ أوردر جديد"**
4. اختار **stage البداية**: 
   - 🎨 **تصميم** = العميل محتاج تصميم جديد
   - 🖨️ **طباعة** = عنده تصميم جاهز ومحتاج طباعة فقط
5. اختار المنتجات، السعر، المحفظة، وعربون (لو فيه)
6. ارفع التصميم لو موجود
7. ✅ احفظ

### تسجيل دفعة عميل
في تبويب "💰 باقي" على لوحتك:
1. اضغط على الأوردر
2. زر **💰 تسجيل دفعة**
3. أدخل المبلغ + المحفظة (اللي العميل دفع فيها)
4. ✅ تأكيد

النظام يحدّث الباقي تلقائياً ويسجّل قيد في الـ ledger.

### استرداد للعميل (refund)
- مش متاح ليك مباشرة — اطلب من **مدير التشغيل** يعتمد طلب استرداد عبر `my-requests.html`

## 📞 المتابعات (Follow-ups)

في panel العميل، تبويب **"📞 متابعات"**:

### إنشاء متابعة جديدة
1. زر **+ متابعة جديدة**
2. اختار النوع: 📞 مكالمة / 💬 واتساب / 📧 إيميل / 🏠 زيارة / 📝 ملاحظة / ⏰ تذكير
3. اكتب ملاحظة
4. اختار **النتيجة**: ✅ ردّ / 📵 لم يردّ / 🎯 مهتم / 🚫 غير مهتم / 🛒 طلب جديد / ⏳ متابعة لاحقاً
5. اختار **تقييم** (1-5 ⭐) لو الـ flow بيستحق
6. حدد **تاريخ المتابعة الجاية** لو محتاج

### إنذار متابعة معلّقة
لو في متابعة بتاريخ مرّ → بتظهر في **قائمة المتابعة** على لوحتك مع badge أحمر.

## 💬 الـ Inbox — `inbox.html`

### المحادثات
- **#عام** = كل الموظفين
- **#خدمة-العملاء** = فريقك
- محادثات مباشرة مع موظفين تانيين

### الـ Stories
- في الأعلى — قصص من فريقك (تصاميم جديدة، تحديثات)
- بتختفي تلقائياً بعد 24 ساعة

### ربط رسالة بأوردر
1. اضغط 📎 + 📦
2. اختار الأوردر من القائمة
3. ابعت — الرسالة مرتبطة بالأوردر دلوقتي

---

## 👀 صلاحياتك (الحقوق)

| تقدر | لا تقدر |
|------|---------|
| ✅ تشوف رقم العميل بالكامل | ❌ تشوف تكلفة المنتج / هامش الربح |
| ✅ تعدّل بيانات العميل | ❌ تحذف أوردر (admin فقط) |
| ✅ تسجّل دفعة من العميل | ❌ تعمل refund مباشرة (طلب → اعتماد) |
| ✅ تربط أوردر بـ inbox | ❌ تعدّل سعر الأوردر بعد تأكيده (محتاج بريف ops_manager) |
| ✅ تشوف كل أوردرات كل العملاء | ❌ تشوف ملفات التصميم (الـ design data للأدوار المعنية) |

---

## ⚠️ أخطاء شائعة

| الخطأ | الحل |
|-------|------|
| دفعة بمبلغ غلط اتسجلت | كلّم مدير التشغيل — يعدّل + يسجّل سبب (audit log) |
| نسيت أعمل follow-up لعميل | متاح ترجع تكتبها بأثر رجعي + اختار التاريخ الفعلي |
| العميل بيقول مش هو اللي طلب | افتح بطاقته + شوف **timeline** الأوردر، فيه كل التعديلات بأسماء |
| الـ inbox مش بيظهر رسالة | عمل refresh (Ctrl+R) — السرعة على الموبايل ممكن تتأخر ثوانٍ |

## 🎯 KPIs بتاعتك (مدير التشغيل بيتابعها)

- **عدد المكالمات** اليومية
- **معدّل الرد** على رسائل inbox خلال X دقيقة
- **عدد الأوردرات** المسجّلة
- **مبلغ التحصيل** اليومي
- **متوسط الـ rating** بتاع الـ follow-ups
